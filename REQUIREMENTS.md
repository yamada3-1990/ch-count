# REQUIREMENTS

## Overview
A web-based character counting application that features real-time character counting, paragraph analysis, and visual space highlighting.

## Core Features
1.  **Character Counting**
    *   Count total characters.
    *   Option to include or exclude newlines (default: configurable, user didn't specify default but said "select if possible").
    *   Count spaces (always counted by default).
    
2.  **Paragraph Counting**
    *   Display character counts per paragraph.
    *   Paragraph definition: Lines separated by newlines (user specified default).
    *   (Optional per user request) User selectable paragraph definition.

3.  **Visuals & UI**
    *   **Tech Stack**: Vite + React.
    *   **Design**: Modern, GUI-friendly, Web App (not exe).
    *   **Space Highlighting**: Real-time highlighting of spaces within the text area.

4.  **Delivery**
    *   Source code pushed to GitHub: `https://github.com/yamada3-1990/ch-count.git`.

## Non-Functional Requirements
*   Performance: Real-time response for typing.
*   Usability: Clear distinction of spaces.
